# 特定商取引法に基づく表記

最終更新: 2026年4月6日

| 項目 | 内容 |
|---|---|
| **販売価格** | アプリ内の購入画面に表示される価格（税込）に準じます。 |
| **支払方法** | Google Play 決済または Apple App Store 決済（各ストアで設定されたお支払い方法） |
| **支払時期** | 購入手続き完了時に即時課金されます。自動更新サブスクリプションの場合、各更新日に課金されます。 |
| **商品の引渡し時期** | 購入手続き完了後、直ちにサービスをご利用いただけます。 |
| **サブスクリプションの解約** | サブスクリプションの解約は、お使いの端末のストア設定から行うことができます。Android: Google Play ストア → お支払いと定期購入 → 定期購入 / iOS: 設定 → Apple ID → サブスクリプション。更新日の24時間前までに解約手続きを行ってください。 |
| **返品・返金** | デジタルコンテンツの性質上、購入後の返品・返金はお受けしておりません。各ストア（Google Play / App Store）のポリシーに基づく返金については、各ストアへ直接お問い合わせください。 |
| **動作環境** | Android 8.0 以上 / iOS 15.0 以上 |
| **販売事業者** | fryx404（個人開発者） |
| **運営責任者** | 古屋 匠 |
| **所在地** | 請求があった場合、遅滞なく開示いたします。本アプリ内「お問い合わせ」よりご連絡ください。 |
| **連絡先** | 本アプリ内の設定 →「このアプリについて」→「お問い合わせ」よりご連絡ください。 |
| **その他** | 本サービスは個人が運営しております。サービスの継続性について最大限努力いたしますが、やむを得ない事情によりサービスを終了する場合は、事前に通知いたします。 |
